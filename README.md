# :electric_plug: EnergIA - Previsão de Consumo de Energia Residencial

Projeto de Inteligência Artificial para prever o consumo de energia elétrica em residências domésticas, utilizando técnicas de **Machine Learning**, com interface web simples para simulação.

---

## :dart: Objetivo

Desenvolver um modelo capaz de prever o **consumo energético (kWh)** com base em características da residência, como:

- Número de moradores  
- Área da residência  
- Uso de ar-condicionado  
- Tipo de construção  
- Outros fatores relevantes  

Além disso, o projeto oferece uma **página web didática** para que qualquer pessoa possa testar o modelo.

---

## :gear: Tecnologias utilizadas

- Python 3.10+
- Pandas
- NumPy
- HTML + CSS
- Mais...

python3 app.py

## :globe_with_meridians: Interface Web

Na página web, o usuário pode informar os dados da residência e obter:

Previsão do consumo em kWh
Classificação do consumo: baixo, médio ou alto
